Pattern Reference

All Manifestation patterns in one place: start direction, stack contract, return value, and a direct scribe link. Signatures are taken from server registrations and use the north-east start direction.

CREATE_LIST_MENU

{ne,awwaqwedwwd}
Menu constructor

Builds and opens a vertical list menu for the caster. It consumes a title and a list of typed UI entries from the stack and sends the resulting menu to the client.

Takes
top -> title
       [entry, entry, ...]
Returns

Pushes nothing. The consumed arguments are removed from the stack.

Notes

Only typed UI iotas are accepted inside the entry list. Invalid individual entries are skipped; a bad outer list mishaps.

CREATE_GRID_MENU

{ne,awwaeawwaqwddad}
Menu constructor

Builds and opens a grid-layout menu for the caster. It works like CREATE_LIST_MENU, but also reads a column count and clamps it into the valid UI range.

Takes
top -> columns
       title
       [entry, entry, ...]
Returns

Pushes nothing. Column count, title, and entry list are consumed in that order from the stack.

Notes

columns must be a number iota. Value is rounded and clamped to 1..10 before build.

UI_BUTTON

{ne,awwaqwedwwdaa}
Entry constructor

Builds a typed button entry for a menu list. Clicking it dispatches stored action iotas in order.

Takes
top -> label
       [action, action, ...]
Returns

Pushes a UiButtonIota(label, actions) onto the stack.

Notes

Action list must be a non-empty list iota. Label can be any iota; display() text is used in UI.

UI_SLIDER

{ne,awwaqwedwwdaw}
Entry constructor

Builds a numeric slider entry with explicit bounds and current value.

Takes
top -> label
       current
       max
       min
Returns

Pushes a UiSliderIota(label, min, max, current) onto the stack.

Notes

min, max, and current must be number iotas. min cannot exceed max, and current must be in range.

UI_DROPDOWN

{ne,awwaqwedwwdawaq}
Entry constructor

Builds a dropdown entry with a fixed options list. Selected index starts at the first item.

Takes
top -> label
       [option, option, ...]
Returns

Pushes a UiDropdownIota(label, options, 0) onto the stack.

Notes

Options slot must be a non-empty list iota. Each option uses its display() text, initial selection is 0.
