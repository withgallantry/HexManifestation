Reference docs for Manifestation, published with GitHub Pages.

I've long suspected a link between media and the mind, but only recently have I begun to understand its shape. When constrained into deliberate patterns, media does not merely represent, it manifests, forming structures that can be perceived... and even interacted with.

More curious still, these constructs seem capable of accepting intent. By imposing order upon them, I can compel them to carry thought forward, translating decision into action, and action into hex.
